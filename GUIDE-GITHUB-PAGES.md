# Publier Volako gratuitement sur GitHub Pages

Vous n'avez pas besoin de Visual Studio Code.

## 1. Créer le dépôt

1. Connectez-vous à https://github.com.
2. Cliquez sur **+**, puis **New repository**.
3. Donnez-lui le nom `volako`.
4. Sélectionnez **Public** pour utiliser GitHub Pages gratuitement.
5. Cliquez sur **Create repository**.

## 2. Envoyer les fichiers

1. Décompressez `Volako-GitHub-Pages.zip` sur votre ordinateur.
2. Dans votre dépôt GitHub, cliquez sur **Add file**, puis **Upload files**.
3. Envoyez les fichiers contenus dans l'archive. `index.html` doit être visible directement à la racine du dépôt et non dans un sous-dossier `volako`.
4. Cliquez sur **Commit changes**.

## 3. Activer GitHub Pages

1. Ouvrez **Settings** dans le dépôt.
2. Dans le menu de gauche, ouvrez **Pages**.
3. Sous **Build and deployment**, choisissez **Deploy from a branch**.
4. Choisissez la branche **main** et le dossier **/(root)**.
5. Cliquez sur **Save**.

Après le déploiement, l'adresse ressemblera à :

`https://VOTRE-NOM.github.io/volako/`

## 4. Installer sur le téléphone

- Android avec Chrome : ouvrez l'adresse, menu **⋮**, puis **Installer l'application** ou **Ajouter à l'écran d'accueil**.
- iPhone avec Safari : ouvrez l'adresse, bouton **Partager**, puis **Sur l'écran d'accueil**.

## Confidentialité

Le code de Volako sera public, mais vos salaires, dépenses, budgets et autres données financières ne seront jamais envoyés dans le dépôt GitHub. Ils restent dans le stockage local du navigateur de chaque téléphone.

Utilisez régulièrement **Paramètres > Exporter une sauvegarde**. Effacer les données du navigateur, désinstaller l'application ou changer de téléphone peut supprimer les informations locales.
