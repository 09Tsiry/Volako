# Volako

Application de gestion financière personnelle, installable sur Android et iPhone.

## Fonctions incluses

- revenus et dépenses en ariary ;
- catégories et historique avec recherche ;
- budgets mensuels ;
- objectifs d'épargne ;
- dettes et prêts ;
- statistiques mensuelles ;
- export et import des données ;
- fonctionnement hors ligne.

## Tester sur ordinateur

Dans le dossier du projet, lancez un serveur local :

```bash
npx serve .
```

Ouvrez ensuite l'adresse affichée dans votre navigateur.

## Installer sur téléphone

- Android/Chrome : menu du navigateur > **Ajouter à l'écran d'accueil** ou **Installer l'application**.
- iPhone/Safari : bouton **Partager** > **Sur l'écran d'accueil**.

Les données sont enregistrées uniquement dans le navigateur de l'appareil. Utilisez régulièrement la fonction **Exporter une sauvegarde**.
